# 元智大學選課機器人 使用說明

## 重要說明

- ✓ 只需下載「元智選課機器人.exe」即可完整使用
- ✓ 無需下載或準備任何其他檔案
- ✓ 無需安裝 Python 或任何依賴套件
- ✓ 完全獨立運作的單一執行檔

## 系統需求

- 作業系統：Windows
- 網路連線

## 使用方式

### 1. 下載檔案

- 從 GitHub Release 頁面下載「元智選課機器人.exe」
- 建議放在容易找到的資料夾中

### 2. 執行程式

- 雙擊「元智選課機器人.exe」
- 首次執行可能需要 10-30 秒載入（解壓縮內部檔案）
- 如果 Windows Defender 出現警告，請點擊「更多資訊」→「仍要執行」

### 3. 填寫資訊

```
帳號：s1234567
密碼：your_password
課程清單（每行一個）：
312,EEB219A
304,CS201B
延遲時間（秒）：2.5
```

### 4. 點擊「開始選課」按鈕

- 程式會自動登入、搜尋課程、嘗試選課
- 執行過程會顯示在下方的「執行記錄」區域

## 課程清單格式

**格式：**`部門代碼,課程代碼+班級`

**範例：**
```
312,EEB219A
304,CS201B
```

**說明：**
- 每行一個課程
- 用逗號分隔部門代碼和課程代碼
- 部門代碼通常是 3 位數字
- 課程代碼包含字母和數字，最後一個字母是班級

### 如何找到部門代碼和課程代碼？

1. 登入元智選課系統
2. 選擇要加選的系所
3. 查看課程列表，記錄「課程代碼」和「系所代碼」

## 延遲時間設定

**建議值：** 2.5 秒

- 延遲時間是每次選課嘗試之間的間隔
- 設定太短可能會被系統判定為異常登入
- 設定太長會降低搶課速度
- 建議保持預設值 2.5 秒

## 常見問題

### Q1：程式執行後沒有反應？

**A1：** 首次執行需要較長時間解壓縮，請耐心等待。如果超過 1 分鐘仍無反應，請確認：
- 防毒軟體是否阻擋了程式
- 是否有足夠的磁碟空間（需要約 500MB）

### Q2：顯示「帳號或密碼錯誤」？

**A2：** 請檢查：
- 帳號是否正確（學號）
- 密碼是否正確
- 是否在選課時間內

### Q3：顯示「選課系統尚未開放」？

**A3：** 請在選課時間內使用本程式。

### Q4：顯示「Wrong coursesList, please check it again!」？

**A4：** 課程清單格式錯誤，請檢查：
- 格式是否為「部門代碼,課程代碼」
- 部門代碼是否正確
- 沒有多餘的空格或符號

### Q5：顯示「異常登入，休息10分鐘!」？

**A5：** 系統偵測到異常登入行為，程式會自動等待 10 分鐘後重試。建議增加延遲時間（例如改成 3-5 秒）。

### Q6：如何停止程式？

**A6：** 直接關閉視窗即可。

### Q7：可以同時選多個課程嗎？

**A7：** 可以，在課程清單中每行輸入一個課程即可。

### Q8：程式會一直重試直到選上課嗎？

**A8：** 是的，對於尚未成功選上的課程，程式會持續嘗試，直到：
- 成功選上
- 顯示「已選過」
- 手動關閉程式

## 注意事項

1. 僅供學習交流使用，請勿濫用
2. 使用本程式仍需遵守學校選課規定
3. 選課成功後請登入選課系統確認
4. 建議在選課開始前先測試程式是否能正常執行
5. 本程式不保證一定能搶到課程
6. 使用本程式的一切後果由使用者自行承擔
